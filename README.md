# Agencia-de-Viagem
📝 Sobre o Projeto
O Sistema de Reservas - Vida Boa na Estrada é um protótipo de aplicação desktop desenvolvido em Java para solucionar um problema real de negócio: a transição de um registro manual (planilhas impressas) para um sistema digitalizado e seguro.

A automação do cadastro inicial de passagens internacionais visa reduzir drasticamente falhas humanas, como erros de digitação e a perda de dados sensíveis de passaportes de clientes.

🎯 Objetivos
Agilidade: Permitir que o atendente registre as informações de forma rápida e segura.

Integridade dos Dados: Captura estruturada de dados essenciais do passageiro.

Consistência Financeira: Cálculo automatizado de tarifas sem dependência de consultas manuais a tabelas externas.

💻 Funcionalidades
Captura de Dados Interativa: Coleta de Nome e Passaporte por meio de caixas de diálogo dinâmicas.

Seleção Estruturada: Menus interativos para escolha de destinos (EUA, Europa, Ásia, Oceania) e classes de viagem.

Motor de Precificação: Lógica que define o valor base pelo destino e aplica multiplicadores conforme a classe escolhida (Econômica: 1.0x, Executiva: 1.5x, Primeira Classe: 2.0x).

Tela de Confirmação: Exibição de um resumo consolidado com todos os dados e o valor total calculado antes da finalização do processo.

🛠️ Tecnologias Utilizadas
Linguagem: Java

Interface Gráfica: Java Swing (JOptionPane)

Paradigma: Programação Orientada a Objetos (POO)
